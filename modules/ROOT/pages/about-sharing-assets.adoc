= アセットの共有

include::_attributes.adoc[]
endif::[]
:imagesdir: ../assets/images/

非公開 Exchange または Exchange ポータルを使用して、Mule XML サンプルコード、テンプレート、コネクタ、API、およびカスタム情報アセットを共有できます。アセットの共有により、組織はコードと API を簡単にアクセスできるようにモデル化して再利用し、リポジトリを提供できます。

非公開 Exchange では、閲覧者、コントリビュータ、または管理者アクセス権を社内開発者に付与することで、アセットをそれらの社内開発者と簡単に共有できます。また、ビジネスグループ内で作成されたアセットは、そのビジネスグループ内の開発者と自動的に共有されます。

Exchange ポータルでは、Exchange 管理者が自分の組織外の開発者とアセットを外部で共有できます。

どちらの Exchange ビューでもユーザがアセットを確認できるため、アセットプロバイダはフィードバックを得ることができます。

詳細は、xref:exchange2-faq.adoc[「FAQ」]を参照してください。
