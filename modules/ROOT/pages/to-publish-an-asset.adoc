= アセットのドラフトコンテンツのパブリッシュ
ifndef::env-site,env-github[]
include::_attributes.adoc[]
endif::[]
:imagesdir: ../assets/images/

アセットを作成すると、アセットの作成を完了するために [Publish (パブリッシュ)] をクリックするように求められます。
その後コントロールが編集メニューに移動し、そこで説明を追加できます。

説明を追加する場合、アセットを利用するユーザにその説明が表示されるように、
アセットのドラフトコンテンツをパブリッシュする必要があります。アセットのドラフト更新をパブリッシュするには、
[Publish (パブリッシュ)] image:ex2-publish.png[ボタン - タスクバー内のパブリッシュ] をクリックします。

このアイコンが表示されていない場合は、[Go To Draft (ドラフトに移動)] をクリックします。

説明を更新したら、タグを追加したり、アセットを共有したり、コピーをダウンロードしたりできます。
