Exchange には、ガバナンス要件に合わせてアクセス管理を設定すると同時に、チーム間のコラボレーションを促進するための複数の方法が用意されています。Anypoint Platform には、[Access Management (アクセス管理)] > [Roles (ロール)] で設定してユーザに異なるアクセスレベルを提供できる、定義済みのロールグループが付属しています。

== Exchange ロールグループ

これらのロールグループは、すべてのビジネスグループレベルに関連します。

* Exchange Viewer (Exchange 閲覧者): このロールグループは、ビジネスグループ内のアセットの表示およびダウンロード権限をユーザに付与。このロールグループのユーザは、新しいアセットの追加、アセットポータルコンテンツの編集、他のユーザとのアセットの共有を行うことはできません。特定のビジネスグループのアセットをコンシュームするユーザにこのロールを割り当てます。

* Exchange Contributor (Exchange コントリビュータ): このロールグループは、ビジネスグループ内のアセットの表示、作成、およびダウンロード権限をユーザに付与。このロールグループのユーザは、既存のアセットバージョンのアセットポータルコンテンツを編集できます。このロールを使用して、アセットポータルの説明を編集および管理するユーザを招待します。

* Exchange Administrator (Exchange 管理者): このロールグループは、ビジネスグループ内のアセットの表示、作成、およびダウンロード権限をユーザに付与。このロールグループのユーザは、Exchange Contributor (Exchange コントリビュータ) と Exchange Viewer (Exchange 閲覧者) ロールが実行可能なすべてを実行できるだけでなく、他のユーザとのアセットの共有およびアセットの削除を行うことができます。このロールを使用して、特定のビジネスグループ内のすべてのアセットに対して Exchange 管理者をセットアップします。

== アセットロール

これらのロールグループは、すべてのアセットレベルに関連します。

これらのロールは、ユーザにビジネスグループレベルで割り当てられたロールグループに加えて、アセットに対してもアクセス権を拡大します。

* Asset Viewer (アセット閲覧者): このロールは、アセットの表示およびダウンロード権限をユーザに付与。このロールのユーザは、アセットポータルコンテンツの編集または他のユーザとのアセットの共有を行うことはできません。このロールを使用して、アセットを表示およびダウンロードするビジネスグループ外のーザを招待します。

* Asset Contributor (アセットコントリビュータ): このロールは、アセットの表示、新規バージョンの追加、およびダウンロード権限をユーザに付与。このロールを使用して、アセットのポータルコンテンツを表示、ダウンロード、追加、および編集するビジネスグループ外のユーザを招待します。

* Asset Administrator (アセット管理者): このロールは、アセットの表示、作成、およびダウンロード権限をユーザに付与。このロールのユーザは、Exchange Administrator (Exchange 管理者) ロールが単一アセットのみに対しして実行可能なすべてを実行できます。このロールは、デフォルトでアセット作成者に割り当てられます。このロールを使用して、アセットに対する管理者権限をその他のユーザにまで拡大するように設定します。Asset Administrator (アセット管理者) はアセットを削除できます。

== Exchange ロールの概要

image::ex2-roles-chart2.png[チャート図 - ロールの表示権限とアクション]

* Exchange Viewer (Exchange 閲覧者) は、ビジネスグループ内の表示権限があり、アセットの表示およびダウンロードが可能。
* Exchange Contributor (Exchange コントリビュータ) は、ビジネスグループ内の表示権限があり、アセットの表示およびダウンロードが可能。このロールは、
アセットの説明を編集することもできます。
* Exchange Administrator (Exchange 管理者) は、ビジネスグループ内の表示権限があり、アセットの表示、ダウンロード、説明の編集、削除、共有が可能。
* Asset Viewer (アセット閲覧者) は、ビジネスグループ内のアセットの表示権限があり、アセットの表示およびダウンロードが可能。
* Asset Contributor (アセットコントリビュータ) は、ビジネスグループ内のアセットの表示権限があり、アセットを表示およびダウンロード可能。このロールは、
アセットのコンテンツを編集することもできます。
* Asset Administrator (アセット管理者) は、ビジネスグループ内のアセットの表示権限があり、アセットの表示、ダウンロード、コンテンツの編集、削除、および共有が可能。
