= API のドキュメント化
ifndef::env-site,env-github[]
include::_attributes.adoc[]
endif::[]
:imagesdir: ../assets/images/

Anypoint Exchange では、API の各機能を説明し、ユーザがさまざまな値の API への送信と出力の受信を試すことができるテスト領域を提供する、API Notebook を作成できます。

API Notebook を作成したら、必要に応じてそのノートが非公開か公開かを制御する API インスタンスを作成できます。ノートのインスタンスを非公開に設定することで、組織内の他のユーザがそのノートを確認できるようになります。準備が整ったら、API Notebook を公開するように API インスタンスを設定できます。

注意: API Notebook は、API インスタンスの使用とは無関係です。
RAML でベース URI を使用しない場合、または API インスタンスを使用しない場合、モッキングサービスを使用してテストデータを生成できます。
